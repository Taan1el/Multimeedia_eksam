# Läbipaistvus ja kasutusõigus

Kõik Moodul 2 pildid on loodud tööriistaga ChatGPT image generation.

Pildid on mõeldud Slow Pour eksamiprojekti veebilehe visuaalseks materjaliks. Failid on dokumenteeritud failis `promptid.md`, kus on kirjas failinimi, kasutatud prompt ja optimeeritud WebP/AVIF väljundid.

Järeltöötlus:

- algsed PNG failid kopeeriti kausta `original/`
- veebikasutuseks loodi WebP ja AVIF versioonid kausta `optimized/`
- sisu ei muudetud, optimeeriti ainult failiformaati
