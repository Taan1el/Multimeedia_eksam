# Eksam Moodul 1 - Veebikujundus

Moodul 1 sisaldab Slow Pour veebilehe kujundust Figmas.

## Sisu

- `figma-link.txt` - Figma faili link
- `algmaterjalid/pildid/` - kujunduses kasutatud pildimaterjal
- `renderdused/` - PNG renderdused desktop, tablet ja mobiil vaadetele
- `.fig` fail - lisatakse pärast Figma eksporti
- logo failid lisatakse pärast käsitsi viimistlust

## Lehed

- Avaleht
- Kohvisordid
- Detailleht
- Kontakt
- Tellimus

## Staatus

Figma failis on loodud disainisüsteemi leht, low-fidelity kavandite raamid ja high-fidelity kavandite raamid desktop, tablet ja mobiil vaadetele.

PNG renderdused on lisatud kausta `renderdused/`.

Enne lõplikku esitamist tuleb lisada klikitav prototüüp, lõplik logo ja eksporditud `.fig` fail.
